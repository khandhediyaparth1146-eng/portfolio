# Khandhediya Parth - Portfolio Website

This is a beautiful, modern single-page portfolio designed for [GitHub Pages](https://pages.github.com/).

## Features
- **Modern Design**: Dark theme with glassmorphic cards and vibrant blue-purple gradients.
- **Interactive**: Smooth scrolling, fade-in animations as you scroll, and a dynamic animated background.
- **Fully Responsive**: Looks great on both desktop and mobile devices.
- **No Complex Build Step**: Built entirely with Vanilla HTML, CSS, and JS. No Node.js or frameworks required, making it extremely easy to host.

## How to Publish on GitHub Pages (Free forever)

1. **Create a new repository** on GitHub named `khandheiya-parth.github.io` (or just name it `portfolio`).
2. **Upload these files**: Upload `index.html`, `style.css`, and `script.js` directly to your new repository.
3. **Enable GitHub Pages**:
   - Go to your repository's **Settings**.
   - Navigate to **Pages** on the left sidebar.
   - Under "Build and deployment", select **Deploy from a branch**.
   - Under "Branch", select `main` (or `master`) and hit **Save**.
4. **Wait a minute**, and your website will be live at `https://[your-github-username].github.io/`

## Local Development

If you want to view the site on your computer before uploading:
Simply double-click `index.html` to open it in your browser!

That's it! Enjoy your stunning new AI/ML Engineer portfolio. 🚀
